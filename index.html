席決め、起家決め

席決め

1.🀀🀁🀂🀃牌を裏返しにし混ぜ、適当な順番で一人ずつ引いていく。

2.🀀を引いた人が、好きな席を指定し、その席を東家として基準になる。

3.他の人は引いた牌の席に着く。

※三麻の場合は🀀🀁🀂牌を使用する。

起家決め

1.🀀を引いた人が仮仮親となってサイコロを振る。

2.出た目の人が仮親となる。

3.仮親になった人がサイコロを振り。

4.出た目の人が起家（最初の親）となります。

※三麻の場合、サイコロの出た目の席が空席の場合は再度サイコロを振り直します。（空席ではない目が出るまで）

四麻 基本ゲームルール

  • 東南戦　25,000点持ちの30,000点返し、喰いタンあり、役の後付けなしのアリナシルールです。
  • 定められた一式の麻雀牌の内、5萬・5筒・5索の各1枚が赤牌となっています。（赤牌の5萬・5筒・5索の各1枚はドラとする）
  • 槓ドラ、裏ドラあり。
  • 一本場は300点です。
  • 形式テンパイありで、テンパイ連荘です。
  • リーチ棒、積棒はアガリ者が取得する。（供託があり、南四局が流局した場合、供託はトップ者に加算する）
  • ダブロン、トリロン無し。頭ハネです。（ロン牌を捨てた人を基準にツモ番が近い人がアガる権利）
  • 得点計算は切り上げ満貫。
  • 南4局終了時、誰も30,000点に満たなかった場合は西場に入り、西4局まで行う。
  • 西4局終了時、誰も30,000点に満たなかった場合は北場に入り、北4局まで行う。
  • 北4局終了時、誰も30,000点に満たなかった場合はトップ者が30,000点になるよう、他3人が足りない点数を割り勘し、トップ者に渡し終了する。
  • ハコテン（持ち点がマイナス）になった時点でゲーム終了。
  • ハコテンになる局の支払いは、全点棒をアガリ者に渡すこと。
    • 例：2,000点持っていて、8,000点を振り込んだ場合は、アガリ者に全点棒の2,000点を支払い、ゲーム終了となる。

途中流局について

（通常の流局と同じで積棒は増えます。親も流れます）

  • 四風連打：1巡目に4人が同じ風牌（東・南・西・北）を連続して捨てると成立
  • 四カン流れ：2人以上が合計4つのカンをした時に成立
    • （4回目のカンで捨て牌がロンされなかった時点で四カン流れとなります。槍槓（チャンカン）があった場合は、四カン流れより優先されます）
  • 九種九牌、4人リーチは続行です。
  • 流し満貫あり
  • オープンリーチなし

フリテン、チョンボ

  • フリテンリーチ、リーチ後の見逃しはツモのみあがれます。ロンはできません。
  • 「同巡内フリテン」

    • 自分がテンパイ（役あり、リーチしていない）していて、他家から自分のアガリ牌が出たときに、その牌でロンをしなかった時点で、「同巡内フリテン」となる。同巡で他の他家がアガリ牌を捨てたとしてもロンすることはできません。同巡内フリテンは自分のツモ番を経て解消されます。

  • チョンボ（誤ロン、役無しアガリ、流局ノーテンリーチ）
    • 【罰符：満貫払い】
    親がチョンボした場合 → 子に4,000点ずつ支払う
    子がチョンボした場合 → 親に4,000点、子に2,000点ずつ支払う
  • チョンボした局はリセットされる。

    • （その局をやり直しする。局は流れません）リーチ棒はリーチ者に戻ります。積み棒はそのまま。連荘数も増えません。

  • 誤チー、誤ポンはお咎めなし。謝って何事もなかったように振る舞いましょう。
  • 多牌、少牌はアガリ放棄。

ポイントの計算

  • ウマ（順位点）あり →【 5-10 】
    • 1位 : +10,000点 （+10P）
    2位 : +5,000点 （+5P）
    3位 : -5,000点 （-5P）
    4位 : -10,000点 （-10P）

    ※100点単位でも同点の場合は上家優先 （「起家から近い方」が優先）です。

  • オカ（トップ賞）あり → 20,000点（20P）

    • ※100点単位でも同点の場合は上家優先 （「起家から近い方」が優先）です。

ポイントの計算方法

1. 原点基準（30,000点）でプラス・マイナスを算出
「ゲーム終了時の持ち点」－「30,000点」でプラス・マイナスを算出します。

2. 点数をポイントに換算する
100点単位は五捨六入で計算します。（500点以下→0点、600点以上→1,000点として扱う）1,000点＝１ポイントで換算する。

3. トップ者にオカを加算する
それぞれのポイントを発表し、トップ者に20ポイントを加算します。

例1：ゲーム終了時、18,500点だった場合の計算方法
1. 18,500点－30,000点 = －11,500点
2. 【－11,500点】 →（五捨六入）→ 【－11,000】 →（ポイント換算）→ 【－11】

例2：ゲーム終了時、55,600点（トップ）だった場合の計算方法
1. 55,600点－30,000点 = 25,600点
2. 【25,600点】 →（五捨六入）→ 【26,000】 →（ポイント換算）→ 【＋26】
3. 26＋20(オカ)＝46 → 【＋46】
